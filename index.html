Macro Meals allows the user to search for healthy food options using custom macronutrient filters. Choose the amount of fat, carbs, and protein you need or just browse through all the meal options. Built using React with an Express backend and MongoDB

Wayfarer is a travel blog allowing the users to set up an account and share their thoughts on a variety of exciting destinations. Built using Python and Django. This was a collaboration of 3 developers

Pets4U is a site designed to inform users of pets available for adoption in their area. Pets can be searched by location and new pets can be added to the site. If you decide to adopt a pet, an email confirmation will be sent to you (note: nodemailer is disabled for the deployed site at this time). This was a collaboration with another developer and it is built using Express and MongoDB
